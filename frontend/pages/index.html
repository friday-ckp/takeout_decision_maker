🗄️ 数据库配置向导

首次使用需配置 MySQL 数据库连接。请在项目 backend/.env 文件中填写以下信息： 

DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的密码
DB_NAME=takeout_decision

配置完成后运行 npm run migrate 初始化数据库，然后重启服务。
